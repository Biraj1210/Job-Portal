# 🚀 Job Portal Web App (Frontend Only)

A modern, responsive **Job Portal Web Application** built using **HTML, CSS, and JavaScript**.
This project simulates a real-world recruitment platform with **student and recruiter roles**, interactive UI, and client-side data management using **localStorage**.

---

## 🌐 Live Demo

👉 https://your-link.vercel.app

---

## 📌 Features

### 👨‍🎓 Student Panel

* User Signup & Login 🔐
* Browse available jobs
* Apply to jobs
* Save jobs for later ❤️
* Search jobs in real-time 🔍
* View analytics (applications & saved jobs) 📊

### 🏢 Recruiter Panel

* Post new jobs
* View all posted jobs
* Simple dashboard interface

---

## ⚙️ How It Works

* User data is stored in **localStorage**
* Jobs posted by recruiters are saved locally
* Students can apply & save jobs
* Analytics are dynamically calculated from stored data

---

## 🚀 How to Run Locally

1. Download or clone the repository

```
git clone https://github.com/your-username/job-portal.git
```

2. Open the folder

3. Run:

```
index.html
```

No installation required ✅

---

## ⚠️ Limitations

* No backend/database (uses localStorage)
* Data is browser-specific (not shared across devices)
* No real authentication security
